## GitHub Pages Hosting

1. Create a new GitHub repository.
2. Upload these files:
   - index.html
   - style.css
   - script.js
   - README.md
3. Go to Settings > Pages.
4. Under Build and deployment, select:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
5. Save.
6. GitHub will give you a website link.
